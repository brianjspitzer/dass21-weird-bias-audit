# DASS-21 WEIRD Bias Audit

This project audits how global mental health app alerts could misclassify users when thresholds are based on WEIRD-heavy internet samples.

Using data from the Open-Source Psychometrics Project, we examine how a global DASS-21 anxiety threshold would flag different demographic groups.

The goal is to clarify what conclusions are safe to draw from large online convenience samples and where interpretation should stop.

## Repository structure

data/     dataset download instructions  
scripts/  analysis scripts  
tables/   generated summary tables  
figures/  visualizations  
docs/     final reproducible report  

## Output

The final deliverable is an RMarkdown memo written for a non-technical audience.
